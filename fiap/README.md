# fiap

Exercicios da pos-graduacao FIAP em Arquitetura e Desenvolvimento Java
(Clean Architecture, SOLID, CQRS/Event Sourcing, sistemas distribuidos,
cloud). Essa fase comeca mais pra frente no cronograma — pasta criada como
placeholder pra nao precisar reestruturar o repo depois.

Quando comecar, decidir se cada fase (mes 1-2, 3-4, 5-6...) vira uma
subpasta ou se o conteudo entra direto em `src/main/java` como os outros
pacotes — depende se os exercicios da FIAP forem coesos o bastante pra virar
codigo Java "normal" ou se forem mais conceituais/arquiteturais (nesse caso,
fica melhor em markdown aqui mesmo, tipo o `appsec-labs/`).
